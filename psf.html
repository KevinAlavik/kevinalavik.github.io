PC Screen Font (PSF) Format Specification

The PC Screen Font (PSF) format is used for console fonts on Linux systems. It has two versions, PSF Version 1 and PSF Version 2. Below are specifications for both versions:

PSF Version 1

The PSF Version 1 header structure is as follows: 

    
    // PSF Version 1 Header
    struct PSFHeaderV1 {
        uint8_t magic[2];   // Magic identifier, should be 0x36, 0x04 for PSF1
        uint8_t mode;       // PSF font mode (0 for PSF1)
        uint8_t charSize;   // Size of each character in bytes
    };

Magic Numbers for PSF Version 1:

  • Magic Number: [0x36, 0x04]

PSF Version 2

The PSF Version 2 header structure is as follows: 

    
    // PSF Version 2 Header
    struct PSFHeaderV2 {
        uint8_t magic[4];   // Magic identifier, should be "psf2" for PSF2
        uint32_t version;   // Version number, should be 0x0200 for PSF2
        uint32_t headerSize; // Size of the header in bytes
        uint32_t flags;      // Flags for additional information (e.g., compression)
        uint32_t length;     // Number of glyphs
        uint32_t charSize;   // Size of each character in bytes
    };

Magic Numbers for PSF Version 2:

  • Magic Identifier: "psf2"
  • Version Number: 0x0200
