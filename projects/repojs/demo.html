Demo Site For repojs

You can init repojs by running the repojs.init() function.

You can fetch and parse + add the data to the document like this:

repojs.repo.fetch("https://puffer.is-a.dev/repos/main-scarlet.json")
‎ .then(data => {
‎ ‎ data = repojs.repo.parse(data, "META");
‎ ‎ repojs.repo.document.append(data, document.body);
‎ });
