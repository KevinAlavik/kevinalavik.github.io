RepoJs

The repo handler package you didnt know you needed

About it

Repojs is a javascript package/library to handle json data easly and it was designed for iOS 3rd party app stores in mind (Web based stores). But it can really be used for any applications that use JSON.

Features

  • Easly manage and handle JSON data
  • Automatic JSON parsing for easy use
  • Support for custom JSON syntax
