Using iTroll To Jailbreak

For checkm8 devices, Written by: @pufferisadev

Requirements

In this tutorial you will need installed:

  • git

Introduction

So there is a new tool in town! iTroll, iTroll is a tool made by me. You can use iTroll to get device information but you can also use it to jailbreak your iDevice. There are a few jailbreak options these are the current ones:

  • Unc0ver
  • Palera1n

In this tutorial we will be using palera1n to jailbreak your checkm8 device.

Supported Devices

  • iPhone 5S
  • iPhone 6, 6S, 6S Plus
  • iPhone 7, 7 Plus
  • iPhone 8, 8 Plus
  • iPhone X
  • iPad 5, 6, 7
  • iPad Pro 2016-2018
  • iPad Mini 2, 3, 4
  • iPad Air 1, 2

Up to iOS 16.5

Installing

First we need to actually download iTroll and for this you will need a Mac or Linux, at the moment iTroll isnt availible for Windows. Start by opening up a terminal and typing in:

git clone https://github.com/kevinalavik/itroll

After that just run this:

cd troll/src

Now we need to install some requirements:

bash install-dep.sh

Now comes the fun part! Now its time to launch iTroll and start jailbreaking

To run it simplly run

python3 main.py
