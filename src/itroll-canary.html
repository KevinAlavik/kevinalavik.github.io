iTroll Canary

Written by: @pufferisadev

Requirements

In this tutorial you will need installed:

  • python3
  • git

Introduction

iTroll Canary: The better version or worst of iTroll, you decide!

Supported Devices

  • Every iOS Device out there! Any iOS Version

Whats new?

We have rewritten the entire iTroll cli handler so now we use click and not our own badly written one. And click makes it easier to add and manage multiple pages so expect easier navigation! We also have a new setup screen (as you see in the cover image).

We have basically rewritten everything.

Options

There is a few options:

We have decided to put things in "categories" so advanced have things like DFU tools (as you can see down below)

We also added "settings" where you can reset iTroll

Jailbreaking!

In version 0.1.2-canary there is two jailbreak options:

  • Unc0ver
  • Palera1n

But (sadly) in version 0.1.2-canary we didn't actually impalement the jailbreak code. But maybe in version 0.1.4-canary we do! So keep your eyes open!

DFU/Recovery Tools

In version 0.1.2-canary there exists options to enter and exit DFU/Recovery mode but, again (sadly) we didn't spend time on implementing the DFU handler. But we will most likely impalement it in version 0.1.4-canary

Installing

Installing just got a little easier!

Start by opening a terminal and running these simple commands:

git clone https://github.com/kevinalavik/itroll -b canary

cd iTroll/src

Now simply run:

python3 main.py

And you should see this screen, after a bit

Then just press enter and iTroll should refresh and now you have successfully installed iTroll Canary!
