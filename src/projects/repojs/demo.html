Demo Site For repojs

You can init repojs by running the repojs.init() function.

You can fetch and parse + add the data to the document like this (this gets all the data from the repo and parses it!):

repojs.repo.fetch("https://puffer.is-a.dev/repos/main-scarlet.json")
‎ .then(data => {
‎ ‎ data = repojs.repo.parse(data);
‎ ‎ repojs.repo.document.append(data, document.body);
‎ });


Demo output:

Parse paramenters passed: data, META.repoName, META.repoIcon (i called the parse function twice). The data variable comes from the repojs.repo.fetch(url) function

The repo passed in is https://puffer.is-a.dev/repos/main-scarlet.json aka my repo full of my apps! (get it here!)
